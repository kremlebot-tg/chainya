# Чайня — сайт чайной

Одностраничный сайт чайной «Чайня» (Москва, ул. Острякова, 3, метро Аэропорт).
Три языка: русский, английский, китайский. Тёмная и светлая темы.

Живой сайт: *(добавить адрес после включения Pages)*

## Как устроено

Весь сайт живёт в одном исходнике — `src.html`. Внутри HTML, CSS и JS без сборщиков,
фреймворков и зависимостей. Тексты на трёх языках лежат в объекте `I18N` в конце файла,
данные о чаях — в `TEA_META` (тип, цена, фото) и в `I18N.<язык>.teas` (названия и описания).

Картинки в исходнике помечены маркером `{{img:имя}}` и подставляются при сборке.

## Сборка

```bash
python3 build.py          # один файл: всё вшито внутрь
python3 build.py --web    # раздельные файлы для хостинга → dist/
```

**`build.py`** — для пересылки: `index.html` (~1,3 МБ), картинки и шрифты вшиты в base64.
Открывается двойным кликом где угодно, работает без интернета и без сервера.

**`build.py --web`** — для сайта: `dist/index.html` (~140 КБ) плюс `dist/img/` и `dist/fonts/`
отдельными файлами. HTML прилетает сразу, ассеты кэшируются браузером.
Именно `dist/` и публикуется.

## Картинки

```bash
./prep_images.sh          # готовит webp из исходных фото
```

Исходники ждёт в `~/Desktop/чайная`. Нужен `cwebp` (`brew install webp`).

Осторожно: `cwebp` выбрасывает EXIF, а `sips` поворот не запекает. Кадры с камеры
(4160×2768 с флагом поворота) приходится доворачивать вручную — за это отвечает
пятый аргумент функции `wide`. Без него фото ложатся на бок.

**`extract_logo.py`** вырезает знак «ЧНЯ» из логотипа, снятого поверх фотографии
плантации: берёт самую большую белую область (плашку), заливкой от краёв отсекает фон,
а всё, куда заливка не дошла, считает знаком. Нужен Pillow. Запускать только если
пришлют новый логотип — результат уже лежит в `src-assets/logo-mark.png`.

## Что где

```
src.html            единственный исходник: разметка, стили, скрипты, тексты
build.py            сборка (два режима)
prep_images.sh      фото → webp
extract_logo.py     знак «ЧНЯ» отдельно от фотоподложки
img/                готовые webp
fonts/              Prata и Golos Text, кириллические и латинские сабсеты
src-assets/         логотип, фавикон, картинка для превью ссылок
dist/               результат сборки для хостинга (не в гите)
```

## Шрифты

Prata (заголовки) и Golos Text (текст) — оба Google Fonts, лицензия OFL.
У них нет иероглифов, поэтому китайская версия рисуется системными шрифтами
(правило `:lang(zh)` в стилях).
